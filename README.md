# Thermo-LIO Website

This is a static website generated from the Thermo-LIO project deck.

## Local Preview

From `D:\isaac-sim`, run:

```powershell
python -m http.server 8080
```

Then open:

```text
http://localhost:8080/site/thermo-lio/
```

## QR Code

The current QR code points to the public website URL:

```text
https://slanzz369.github.io/thermo-lio/
```

If the website URL changes, regenerate `assets/thermo-lio-qr.png` with the new URL and update `PUBLIC_SITE_URL` in `script.js`.
